OPTION A

Docker Setup

Containerized environment for rapid deployment.

git clone https://github.com/scholarsync/mcp-server.git
cp .env.example .env
docker-compose up -d
OPTION B

Local Python

Direct execution on your host machine.

pip install -r requirements.txt
export MONGO_URI="mongodb+srv://..."
python server.py
OPTION C

Remote HTTP/SSE

Connect via Secure Server Events. Ideal for distributed teams and centralized compute clusters.

GET /connect?protocol=sse

Claude Desktop Configuration

Update your claude_desktop_config.json to enable ScholarSync tools.

{
  "mcpServers": {
    "scholarsync": {
      "command": "python",
      "args": ["/path/to/scholarsync/server.py"],
      "env": {
        "MONGODB_URI": "YOUR_CONNECTION_STRING",
        "LOG_LEVEL": "DEBUG"
      }
    }
  }
}
